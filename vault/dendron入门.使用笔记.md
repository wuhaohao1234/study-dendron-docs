# 使用笔记

## 前提

vscode安装Dendron插件

已经学会markdown语法

## 初始化工程目录

ctrl + shift + p

输入Dendron: Initialize Workspace

## 新建文档

ctrl + shift + j

输入文档名称

[[test]]

删除文档

ctrl + shift + d

## 创建层次结构

ctrl + shift + j

test.world


dendron中的层次结构只是.定界文件。

可以点击编辑器右边的dendron tree view查看层级结构

## 查找文件

这里使用ctrl + shift + j 输入文件名，支持模糊查询
